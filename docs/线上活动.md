### 独立开发者长期陪伴群

👋 独立开发者长期陪伴群，在这里每周打卡，帮助彼此完成项目，交换灵感，认识朋友

🌱 社群规则：
- 公开成长：践行「Build in Public」精神，边开发边分享。
- 每周打卡：每周至少分享一次自己正在做的项目的进度/计划/经验（有实质内容），否则会被移出群聊，不定期清理僵尸成员。
- 互助共进：积极互动，成为彼此的首批用户，收集宝贵的意见，共享资源。
- 灵感碰撞：每周六晚上 9 点线上腾讯视频会议，交流碰撞 Idea: https://meeting.tencent.com/dm/kKms79aQxAvf
- 共创文档：打卡（有实质内容）两次后可申请加入共创文档: https://hackathonweekly.feishu.cn/wiki/XOu1wqbg4iFd6SkBN9UcQVWun4N

🌈 加入条件：
- **必须**有正在进行中的独立开发项目，承诺每周打卡一次。

🌈 打卡格式：
```
#打卡
时间：2024-05-16
名称：john
自我介绍：全栈工程师
阶段目标：6月底上架一款AI工具网站
近期进度：上周基于模板搭建了站点
近期计划：本周确定产品的实现逻辑
链接：在这里放上你的 build in public 链接，比如社交媒体账号，产品介绍等。
思考：在这里写一些你几天的所思所想
```

📝 加入方式：
填写报名表: https://hackathonweekly.feishu.cn/share/base/form/shrcn8CyBVCktjBkaOmJsRZDLVd
如果审核通过，24小时内会与您联系，我们期待每一个独特的你！
欢迎关注公众号 hackathon_weekly 或者我们的网站 hackathonweekly.com

欢迎新小伙伴们，欢迎参加此次五一线上独立开发陪伴打卡群。
5.1 到 5.5 的时间内，每位小伙伴需要在群上按如下格式写下自己五一的具体目标和计划。
5.1 晚上 24 点前，如果没有发送的小伙伴会被请出群聊哦。

```
名称：alex
自我介绍：算法工程师；略懂前端后端；
时间：2024-05-01
阶段目标：5天内上架一款智能语音工具网站
今日进度：初始化项目；构思产品形式
明日计划：确定每个模块的功能和实现方式
思考：在这里写一些你的所思所想。

```
