### 独立开发者长期陪伴群

👋 独立开发者朋友们，你是否正孤身一人砥砺前行？

想象一下，有一个地方，你可以放心分享你的独立开发的点滴，收获即时的反馈和鼓励；一个社群，让你的每个灵感瞬间都能遇见共鸣，每个技术难题都能找到同行者的解答。这就是「独立开发者陪伴群」——一个温馨的互助平台，专为像你这样的创新者打造。

🌱 我们的愿景：
- 公开成长：践行「Build in Public」精神，边开发边分享
- 互助共进：每周打卡，不仅是记录，更是交流。链接你的开发日志，让我们一起见证你的项目从 0 到 1 的飞跃。
- 免费公益：不会有任何收费，保持公益。
- 实战反馈：成为彼此的首批用户，从真实体验中收获宝贵意见，助力产品迭代升级。
- 灵感碰撞：每周六晚上 8 点线上腾讯视频会议，交流碰撞 Idea.
- 专属社群空间：一个活跃且友好的交流环境，让孤独的开发之路不再孤单。

🌈 招募条件：
- **必须**有正在进行中的独立开发项目，每周在群上分享至少 2 次开发进度 (打卡)。
- 承诺积极分享互动，记录进度，分享思考，共享资源。

⚙️社群规则：
- Build In Public：经常性在群上分享自己的独立开发进展 (打卡)，包括遇到的问题、解决办法、成果展示等，你可以附上作品/博客/社交媒体链接。
- 一周打卡小于 2 次 或者 连续 7 天没有任何互动会被移出社群（无意义互动或打卡会被识别）
- 禁止广告（除了在打卡中宣传自己开发中的作品）

📝 加入方式：
扫码添加微信，请主动介绍一下你正在开发中的项目（具体目标和计划）以及你的 github 主页链接，审核通过后会拉你进入群聊，我们期待每一个独特的你！
