## 网络上的争论很多时候是信息不对称带来的

包容对待评论，尝试去理解他们说出这些话的原因，而不是因为这些话而生气

一个有趣的例子是，有个开源作者发布了免费的软件到 Github 上并公开了自己的微信联系方式，但一堆时间后，他发现微信上常常来一些"无理取闹的用户", 指手划脚地要求他提供各种"售后服务", 后面他才发现他的软件被当作付费软件在淘宝上售卖了，他的微信号被当作售后服务的联系方式。

有很多小白用户，他们不知道开源软件的概念，也不知道作者是免费提供的，他们只知道在淘宝上买了一个软件，但是软件有问题，他们就会找到作者的联系方式，要求作者提供售后服务。

其实互联网上的很多评论也是如此的，互联网上没办法面对面交流，信息不对称，就会导致很多误解和争论，包容一些，理性平和地看待，没有必要因此愤怒。
