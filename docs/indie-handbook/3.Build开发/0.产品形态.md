## 采用什么样的产品形态

当你的想法准备的差不多，接下来肯定要想，采用什么样的产品形态。就目前来说，已知的产品形态包括：

- Web站点
  - 这种是最传统，也是最简单的，也是见的最多的产品形态了。用户打开一个web站点，来使用相应的服务
- App应用
  - 这种是最流行，使用范围最广的产品形态。用户打开手机应用，来使用相应服务。在这个领域，国内与国外，苹果与安卓又有很多差异。
- 第三方小程序
  - 这个在中国国内比较流行，比如微信小程序，抖音小程序等等。
- 浏览器插件
  - 目前Chrome浏览器的市场份额很高，很多开发者为Chrome浏览器开发插件，以此来赚取利润。
- 其他平台插件, 这个有很多, 比如
    - wordpress插件
    - Shopify插件
    - 企业微信插件
    - 钉钉插件
    - canvas插件
    - figma插件
- 电商平台
  - 国内的淘宝，京东，拼多多，国外的亚马逊，eBay等等
- 社交平台
  - 微信，微博，抖音，Facebook，Twitter等等, 比如抖音,微信小红书都可以开店
  - 你甚至可以在 Discord 上开始你的产品, 比如 Midjourney 就是这么干的
- 游戏
  - Stream平台, Epic平台, 当然还有很多其他方式
