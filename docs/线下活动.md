## 一、长期活动

目前只有深圳线下活动，其他城市的欢迎自发组织活动，我们会提供活动宣传支持。
- 每周一次疯狂星期日 / 疯狂代码日
  - 极客茶话会 1h: 聊聊天，碰碰idea
  - 聚众写代码 一个下午或一整天: 带上你的电脑，和其他小伙伴快速做出一个 mvp

### 每周一次的极客茶话会

📅时间：每周末一小时线下聚会
📍地点：不同城市的咖啡厅
🌈活动特色：
- 主题：AI / 互联网 / 出海 / 极客
- 5 分钟项目介绍：每人 5 分钟分享正在做的项目或是酝酿中的奇思妙想亦或是一周所见所闻。
- 自由碰撞：分享后开放提问与讨论，让不同的视角激发出更多创意火花。
- 欢迎小伙伴带着电脑过来碰撞 idea，活动结束后可以继续自由聊天协作。
